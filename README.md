# AI仮想投資シミュレーター

> ⚠️ **仮想シミュレーション専用** — 実際の証券口座への接続・実注文は一切行いません

過去の株価データをもとに「6ヶ月前にこのポートフォリオを組んでいたら」という仮定シナリオを可視化する、教育・学習目的の仮想投資シミュレーターです。

---

## 画面

| シミュレーター | 使い方ガイド |
|---|---|
| `http://127.0.0.1:8080` | `http://127.0.0.1:8080/guide` |

---

## 機能

- 📊 日本株候補10銘柄を自動選定（選定理由付き）
- 💹 仮想1,000万円で買い・売りシミュレーション
- 📈 過去6ヶ月のポートフォリオ推移グラフ
- 🏆 勝率・最大利益/損失銘柄・平均損益率を自動計算
- 💾 SQLiteに売買履歴・判断理由・損益を永続保存
- 🌐 起動時にSafari（解説）とChrome（本体）を自動で開く

---

## 起動方法

```bash
# 1. 初回・リセット時のみ（約30〜60秒）
python3 auto_invest.py

# 2. サーバー起動（次回以降はこれだけ）
python3 app.py
```

起動すると自動的に
- **Safari** → `http://127.0.0.1:8080/guide`（使い方ガイド）
- **Chrome** → `http://127.0.0.1:8080`（シミュレーター本体）

が開きます。左右に並べてご利用ください。

## 免責事項

- このアプリは**教育・学習目的の仮想シミュレーター**です
- 実際の証券口座への接続・実注文は**一切行いません**
- 表示される損益・候補銘柄は過去データに基づく仮定であり、**将来の運用成績を保証するものではありません**
- 実際の投資判断・売買には**絶対に使用しないでください**
