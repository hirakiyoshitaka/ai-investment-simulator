仮想の1,000万円を使って日本株をシミュレーションする方法をわかりやすく解説します

⚠️ 仮想シミュレーション専用 — 実際の売買は一切行いません

起動方法（ステップ別）
🖥️ ターミナルでの操作手順
📁

STEP 1 — プロジェクトフォルダに移動

cd ~/AI投資プログラム # フォルダに移動します
⚡

STEP 2 — ポートフォリオを初期化（初回・リセット時のみ）

python3 auto_invest.py # 実行すると以下が行われます： # ・10銘柄を6ヶ月前の実株価で仮想購入 # ・過去6ヶ月分のパフォーマンスデータを計算・保存 # ・SQLiteデータベースに全データを保存 # # ⚠️ 完了まで約30〜60秒かかります（株価取得のため）
ポイント： 初回だけ実行すれば、以降はサーバー起動のみでOKです。 ポートフォリオをリセットしたい時だけ再実行してください。
🚀

STEP 3 — サーバーを起動

python3 app.py # # 起動成功の確認メッセージ： # ======================================================= # AI仮想投資シミュレーター 起動中... # ブラウザ: http://127.0.0.1:8080 # =======================================================
🌐

STEP 4 — ブラウザで開く

# 下記URLをブラウザに入力（またはコマンドで開く）： http://127.0.0.1:8080 # Macの場合はこのコマンドで自動的に開きます： open http://127.0.0.1:8080
🛑

STEP 5 — サーバーを止める時

# ターミナルで Ctrl + C を押す ^C

仮想売りの操作手順
🔴 仮想で株を「売る」手順
【仮想売り — 操作フロー】 方法A：保有銘柄テーブルから直接売却（簡単） ① 保有銘柄テーブルで売りたい銘柄の行を確認 │ ▼ ② 右端の 売却 ボタンをクリック │ ▼ ③ 「何株売りますか？（保有: XX株）」とダイアログが表示 │ ▼ ④ 売却株数を入力（例: 20）→ OKをクリック │ ▼ ✓ 自動的に現在の株価で仮想売却が実行される ✓ 仮想現金残高が増加し、実現損益がSQLiteに保存される 方法B：トレードパネルから売却 ① 銘柄コードを検索して株価を確認 ② 株数を入力 ③ 仮想売り ボタンをクリック

よくある質問
質問 回答
💰 仮想現金が足りなくなったら？ 「リセット」ボタンで¥1,000万からやり直せます。または保有株を売却して資金を作ってください。
📊 株価が古い（🟡マークが表示） トレードパネル下部の「株価を最新に更新」ボタンを押してください。約8〜15秒で更新されます。
📈 グラフが表示されない auto_invest.pyを再実行してください。パフォーマンスデータが計算・保存されるとグラフが表示されます。
🔄 ポートフォリオをリセットしたい 画面右上の「リセット」ボタンを押すと全データが削除されます。その後auto_invest.pyを再実行してください。
🏦 実際の口座に注文は出る？ 絶対に出ません。このアプリはyfinanceで株価を取得するだけで、証券会社への接続機能は一切ありません。
📝 取引履歴はどこに保存される？ プロジェクトフォルダのportfolio.db（SQLite）に永久保存されます。再起動後も消えません。
🌐 インターネットは必要？ 株価取得（yfinance）にインターネット接続が必要です。オフラインの場合は最後に取得した価格を表示します。
⚡ 独自銘柄を追加したい トレードパネルで任意の4桁コードを入力して検索できます（例: 7203 = トヨタ、9984 = ソフトバンクG）。

⚠️ 重要な注意事項（免責事項）

  • 🔴 このアプリは教育・学習目的の仮想シミュレーターです
  • 🔴 実際の証券口座への接続・実注文は一切行いません
  • 🔴 表示される損益・候補銘柄は過去データに基づく仮定であり、将来の運用成績を保証するものではありません
  • 🔴 実際の投資判断・売買には絶対に使用しないでください
  • 🔴 投資にはリスクが伴います。実際の投資は証券会社・金融アドバイザーに相談してください
